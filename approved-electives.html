CSCI 4300 - Web Programming

This class involved learning HTML, CSS, Bootstrap, Javascript, SQL, and python through assignments and implementing all of these fundamental components to into a final project that produced an interactive, dynamically-generated, and database-enabled web pages.

CSCI 4050 - Software Engineering

This class consisted of two simultaneous parts where one included learning the full cycle of a software system development effort, including requirements definition, system analysis, design, implementation, and testing. The second part involved a team semester project that revolved around developing a movie-booking online service while following the full software life cycle.
